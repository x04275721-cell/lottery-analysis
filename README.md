# 彩票预测系统 V6.0

体彩排列三 / 福彩3D 预测系统

## 功能

- **五码推荐**：基于位置频率统计
- **金银胆**：冷热号码分析
- **四注号**：多条件筛选
- **和值预测**：统计近期和值出现频率
- **历史数据**：查看历史开奖记录
- **回测验证**：验证预测命中率

## 定时任务

系统每天23:00（北京时间）自动更新数据，通过GitHub Actions执行。

## 文件说明

- `fetch_history.py` - 获取历史开奖数据
- `daily_conditions.py` - 生成每日推荐
- `lottery_v6.py` - 主预测算法
- `backtest_v6.py` - 回测验证
- `update_all.py` - 每日更新总控
- `hezx_sum.py` - 和值预测模块

## 预测方法说明

本系统基于历史数据的统计分析，使用9种方法组合：
1. 奇偶比统计
2. 大小比统计
3. 质合比统计
4. 和值尾统计
5. 跨度统计
6. 邻孤传统计
7. 复隔中统计
8. 热温冷统计
9. 和值统计

**重要提示**：彩票是完全随机事件，历史数据统计只能提供参考，不能真正预测未来。
