# 彩票分解式项目记忆

## 核心规则（已确认）

**分解式正确判断规则**：
- 正确：两边都有不同数字（1+2、2+1、1+1）
- 错误：只有一边有不同数字（3+0、0+3、2+0、0+2）
- 注意：组三要看不同数字的分布，不是出现次数
- "单边出3"的情况：指开奖号的3个不同数字全部落在某一组（4码组出3或6码组出3、对码组出3或剩余组出3），这是错误情况

## 两种分解方法

1. **对码法**：利用差5对码 (0-5, 1-6, 2-7, 3-8, 4-9)
2. **和值尾四六分解**：根据上期和值尾查表

**和值尾对照表**：
- 0/5 → 4码组:0126, 6码组:345789
- 1/6 → 4码组:1237, 6码组:045689
- 2/7 → 4码组:2348, 6码组:015679
- 3/8 → 4码组:3459, 6码组:012678
- 4/9 → 4码组:0456, 6码组:123789

## 组合使用

- 交集 = 对码组 ∩ 4码组
- 只有当两种方法都正确时，才算组合正确

## 稳定度检查

- 回测最近30期
- 正确率 >= 85%：稳定，可放心使用
- 正确率 < 85%：显示 [WARN] 近期不太稳定，谨慎使用

## 334断组方法（学习进行中）

### 核心原理

- 假设：三个奖号同时出现在三个不同组（1-1-1分布）的概率较低
- 判定：奖号只出现在1组或2组 = 断组成功；1-1-1分布 = 断组失败

### 方法一：对码断组法（已学）

1. 取上期奖号，找每个数字的差5对码 (0-5, 1-6, 2-7, 3-8, 4-9)
2. 合并奖号与对码，去重后取4个作为第一组（4码组）
3. 剩余6个数字平均拆分为两个3码组
4. 顺序无所谓，只要不重不漏即可
